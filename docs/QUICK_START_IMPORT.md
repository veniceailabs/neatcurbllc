# Quick Start: Import Your Yardbook Data

This quick start guide will get you up and running in 5 minutes.

## Step 1: Export from Yardbook (5 minutes)

Go to your Yardbook account and export 4 CSV files:

1. **Customers/Clients** - All your customer contacts
2. **Invoices** - All your invoices
3. **Payments** - All your payment records
4. **Expenses** - All your business expenses

Save them with these names:
- `customers.csv`
- `invoices.csv`
- `payments.csv`
- `expenses.csv`

## Step 2: Log into Neat Curb Admin

1. Go to your Neat Curb admin dashboard
2. Log in with your admin credentials

## Step 3: Import Your Data (10 minutes)

### Import Customers First

1. Click **Import Data** in the left sidebar
2. Click **Import Customers**
3. Click the file upload area
4. Select your `customers.csv` file
5. Click **Parse CSV**
6. Review the preview
7. Click **Import X Records**
8. Wait for success message ✓

### Import Invoices

1. Click **Import Data** again (or stay on the page)
2. Click **Import Invoices**
3. Select your `invoices.csv` file
4. Click **Parse CSV** → **Import X Records**
5. Wait for success message ✓

### Import Payments

1. Click **Import Payments**
2. Select your `payments.csv` file
3. Click **Parse CSV** → **Import X Records**
4. Wait for success message ✓

### Import Expenses

1. Click **Import Expenses**
2. Select your `expenses.csv` file
3. Click **Parse CSV** → **Import X Records**
4. Wait for success message ✓

## Step 4: Verify Your Data

After importing, check each section:

1. **Customers** - Click **Clients** in sidebar
   - Should see your imported customer names
   - Count should match your import count

2. **Invoices** - Click **Invoices** in sidebar
   - Should see your invoice amounts and dates

3. **Expenses** - Click **Expenses** in sidebar
   - Should see your expense items
   - Totals should match your records

## That's It! 🎉

You've successfully migrated from Yardbook to Neat Curb!

## Next Steps

### Set Up Your Services

1. Click **NeatCurbOS** in the sidebar
2. Scroll to "1) Services & Prices"
3. Add your common services (e.g., "Driveway Snow - Small", "Driveway Snow - Large")
4. This makes quoting faster

### Create a Test Quote

1. Go to **NeatCurbOS**
2. Scroll to "2) Quote Builder"
3. Select a customer from your imported list
4. Create a test quote
5. This confirms everything is working

### Check Your Settings

1. Click **Settings** in sidebar
2. Review your business info
3. Update anything that's not correct

## Troubleshooting

### Import doesn't work
- Make sure you're using CSV format (not Excel or PDF)
- Check that the first row has column headers
- Try importing a smaller test file first

### Data looks wrong
- Check that the imported amounts look correct
- Verify customer names match your records
- Look at the import history to see what was imported

### Missing the Import Data page
- Log out and log back in
- Clear your browser cache
- Try a different browser

## Need Help?

If something goes wrong:

1. Check the error message - it usually tells you what's wrong
2. Re-export the CSV from Yardbook
3. Try the import again
4. Contact support with your CSV file

## Important Notes

- ✓ Duplicates are automatically skipped
- ✓ All imports are logged in the Import History
- ✓ You can import the same file multiple times (duplicates won't create new records)
- ✓ Your original Yardbook data stays intact - this is just a copy
